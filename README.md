
Usage
=====

Click [here](http://reaktor.github.com/learn-rx/start.html)  or this and point your browser to file://.../start.html


Commands

    next slide: ctrl-. 
    prev slide: ctrl-, 
    splitview:  ctrl-2
    pen:        ctrl + mouse

Some demos require a server: 

    cd server
    sbt update
    sbt jetty-run
